# Awesome 3D Game - Hero Sections\n\nA modern React 18 + TypeScript + Vite application featuring three premium hero landing page designs: **Lithos**, **Velorah**, and **Mainframe**.\n\n## 🎯 Features\n\n### Three Premium Hero Designs\n\n#### 🪨 **Lithos** - Geology Brand\n- **Cursor-following spotlight reveal effect** with canvas-based soft mask\n- Dark-themed with elegant animations\n- Responsive typography (5xl → 8xl)\n- Interactive \"Start Digging\" call-to-action\n- Smooth easing with Ken Burns zoom effect\n\n#### ✨ **Velorah** - Cinematic Studio\n- Fullscreen looping video background\n- Glassmorphic navigation with liquid glass effect\n- Minimalist, vertically-centered composition\n- Fade-rise animations with staggered delays\n- Responsive mobile menu with hamburger toggle\n\n#### 🎬 **Mainframe** - Creative Agency\n- **Mouse-scrubbed video control** - scrub left/right to navigate video timeline\n- Typewriter animation with blinking cursor\n- A.R.I.A (Adaptive Response Interface Agent) branding\n- Action pill buttons (white, outline, email copy)\n- Animated button appearance with transitioned opacity\n\n### Modern Tech Stack\n\n- **React 18** with hooks\n- **TypeScript** for type safety\n- **Vite** for lightning-fast builds\n- **Tailwind CSS** for utility-first styling\n- **Custom animations** (CSS keyframes)\n- **Error Boundary** for graceful error handling\n- **Page Transitions** for smooth hero switching\n- **Accessibility features** (ARIA labels, focus rings, semantic HTML)\n\n## 🚀 Getting Started\n\n### Prerequisites\n\n- Node.js 16+ \n- npm or yarn\n\n### Installation\n\n```bash\n# Clone and install dependencies\ngit clone https://github.com/omerfaisalraza-cyber/awesome-3d-game.git\ncd awesome-3d-game\nnpm install\n```\n\n### Development\n\n```bash\n# Start dev server (opens on http://localhost:5173)\nnpm run dev\n```\n\n### Production Build\n\n```bash\n# Build for production\nnpm run build\n\n# Preview build locally\nnpm run preview\n```\n\n## 📂 Project Structure\n\n```\nsrc/\n├── components/\n│   ├── Lithos.tsx           # Geology brand with spotlight reveal\n│   ├── Velorah.tsx          # Video + liquid glass hero\n│   ├── Mainframe.tsx        # Video scrub + typewriter hero\n│   ├── PageTransition.tsx   # Smooth transitions between heroes\n│   └── ErrorBoundary.tsx    # Error handling wrapper\n├──         # Global styles & animations\n```\n\n## ✨ Key Components\n\n### Lithos\n\n**Features:**\n- Real-time cursor tracking with smoothing\n- Canvas-based radial gradient mask\n- Two-layer image reveal system\n- Staggered text animations\n\n**Customization:**\n```typescript\nconst SPOTLIGHT_R = 260;  // Spotlight radius\nconst BG_IMAGE_1 = '...'; // Base image URL\nconst BG_IMAGE_2 = '...'; // Reveal image URL\n```\n\n### Velorah\n\n**Features:**\n- Auto-playing looping video\n- Liquid glass effect (CSS blend modes + backdrop filter)\n- Mobile-responsive navigation\n- Smooth fade-rise Typewriter effect with configurable speed\n- Copy-to-clipboard button with visual feedback\n- Animated pill buttons\n\n**Customization:**\n```typescript\nconst SENSITIVITY = 0.8;   // Video scrub recommended)\n\n### Animations not playing?\n- Ensure `prefers-reduced-motion: reduce` is not set in system preferences\n- Check browser DevTools for CSS errors\n\n### Typewriter not working?\n- Verify `useTypewriter` hook is imported correctly\n- Check that `startDelay` and `speed` are in milliseconds\n\n### Canvas mask not showing?\n- Ensure both base and reveal images are loaded\n- Check browser console for any errors\n- Verify canvas dimensions match viewport\n\n## 🤝 Contributing\n\nFeel free to fork, modify, and customize these components for your own projects!\n\n---\n\n**Made with
